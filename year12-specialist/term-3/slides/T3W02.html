Lesson 1 · 13A

Vector functions and paths

  • What \(\mathbf{r}(t) = x(t)\mathbf{i} + y(t)\mathbf{j}\) means, and what its path is.
  • Eliminate the parameter to find the Cartesian equation — and keep the domain honest.
  • The circular/elliptical pattern: divide, square, add.

From vector function to Cartesian path

Definition

\(\mathbf{r}(t) = x(t)\mathbf{i} + y(t)\mathbf{j}\) assigns a position to each time \(t\). The set of points traced out is the path.

  • To find the Cartesian equation: write \(x = x(t)\), \(y = y(t)\), then eliminate the parameter \(t\).
  • Keep track of the domain: the path may be only part of the Cartesian curve.
  • A vector function carries more information than its path: where, but also when, and in which direction.
Three-part answer

Describing a path fully = Cartesian equation + restriction on \(x\) (or \(y\)) + direction of travel and start point.

Lesson 1 · Vector functionsVCE Specialist Mathematics 3&4

Worked example: substitute out the parameter

Example 1

Find the Cartesian path of  (a) \(\mathbf{r}(t) = (2t+1)\,\mathbf{i} + (t-2)\,\mathbf{j}\), \(t \in \mathbb{R}\)   (b) \(\mathbf{r}(t) = t\,\mathbf{i} + t^2\,\mathbf{j}\), \(t \geq 0\).

  • (a) From \(y = t - 2\): \(t = y + 2\). Substitute: \(x = 2(y+2) + 1 = 2y + 5\) — a straight line, all of it (no restriction, since \(t\) takes every real value).
  • (b) \(x = t\), \(y = t^2 \Rightarrow y = x^2\) — but \(t \geq 0\) forces \(x \geq 0\): only the right half of the parabola, traced left-to-right from the origin.
  • Same curve \(y = x^2\), different vector function \(\mathbf{r}(t) = t^2\mathbf{i} + t^4\mathbf{j}\)? Also \(x \ge 0\) — but it never returns: many vector functions share one path.
Habit

Always ask: which piece of the curve, and which way along it?

Worked example: circular and elliptical paths

Example 2

Describe the path of \(\mathbf{r}(t) = 2\cos t\,\mathbf{i} + 3\sin t\,\mathbf{j}\), \(t \geq 0\).

  • \(\cos t = \dfrac{x}{2}\), \(\sin t = \dfrac{y}{3}\); Pythagoras gives \(\dfrac{x^2}{4} + \dfrac{y^2}{9} = 1\).
  • An ellipse, centre \(O\), semi-axes \(2\) and \(3\); starts at \((2, 0)\), traced anticlockwise, period \(2\pi\).
  • Pattern: \(a\cos t\), \(b\sin t\) → divide, square, add — \(\cos^2 t + \sin^2 t = 1\) eliminates \(t\).
Direction check

Just after \(t = 0\): \(x = 2\cos t\) decreasing, \(y = 3\sin t\) increasing — moving up and to the left, so anticlockwise.

Seeing it: \(\mathbf{r}(t) = 2\cos t\,\mathbf{i} + 3\sin t\,\mathbf{j}\)

The path is \(\dfrac{x^2}{4} + \dfrac{y^2}{9} = 1\); the vector function also tells us the start \((2,0)\) and the anticlockwise direction.

Lesson 2 · 13B

Where — and when

  • Read positions off \(\mathbf{r}(t)\): axis crossings, particular points, particular times.
  • The crucial distinction: paths crossing vs particles colliding.
  • Set up and solve simultaneous component equations.

Worked example: reading the motion

Example 3

\(\mathbf{r}(t) = (t-2)\,\mathbf{i} + (t^2 - 4)\,\mathbf{j}\) for \(t \geq 0\). When and where does the particle cross the \(x\)-axis?

  • Crossing the \(x\)-axis means the \(\mathbf{j}\)-component is zero: \(t^2 - 4 = 0 \Rightarrow t = 2\) (reject \(t = -2\): outside domain).
  • Position at \(t = 2\): \(\mathbf{r}(2) = 0\,\mathbf{i} + 0\,\mathbf{j}\) — it passes through the origin (so it crosses both axes at once here).
  • Translation table: on the \(x\)-axis ⇔ \(y\)-component \(= 0\); on the \(y\)-axis ⇔ \(x\)-component \(= 0\); at point \((a,b)\) ⇔ both components match simultaneously.
Two different questions

"Does the path pass through \(P\)?" is about the curve. "Do two particles collide?" needs the same position at the same time.

Lesson 2 · Where and whenVCE Specialist Mathematics 3&4

Worked example: do they collide?

Example 4

Particles \(A\) and \(B\) move with \(\mathbf{r}_A(t) = (2t+3)\,\mathbf{i} + (t+2)\,\mathbf{j}\) and \(\mathbf{r}_B(t) = (t+5)\,\mathbf{i} + 2t\,\mathbf{j}\), \(t \geq 0\). Do they collide? If so, when and where?

  • Collision needs both components equal at the same \(t\). Equate \(\mathbf{i}\): \(2t + 3 = t + 5 \Rightarrow t = 2\).
  • Check \(\mathbf{j}\) at \(t = 2\): \(A\) has \(t + 2 = 4\); \(B\) has \(2t = 4\). ✓ Both conditions hold.
  • They collide at \(t = 2\), at the point \((7, 4)\).
  • If the \(\mathbf{j}\)-check had failed, the answer would be "no collision" — even if the two straight-line paths still cross somewhere.

Seeing it: same point, same time

Dots mark \(t = 0, 1, 2, 3\) on each path. At \(t = 2\) the dots coincide at \((7,4)\): a collision, not just a crossing.

Particles \(P\) and \(Q\) have \(\mathbf{r}_P(t) = t\,\mathbf{i} + (2t-2)\,\mathbf{j}\) and \(\mathbf{r}_Q(t) = (t-1)\,\mathbf{i} + t\,\mathbf{j}\), \(t \geq 0\).

Lesson 3 · 13C

Calculus of vector functions

  • Differentiate and antidifferentiate componentwise; the constant of integration is a vector.
  • \(\dot{\mathbf{r}}(t)\) is tangent to the path — and \(\dfrac{dy}{dx} = \dfrac{\dot y}{\dot x}\).
  • Build \(\mathbf{r}(t)\) from \(\ddot{\mathbf{r}}(t)\) and initial conditions.

Differentiating componentwise

\[\dot{\mathbf{r}}(t) = \frac{dx}{dt}\,\mathbf{i} + \frac{dy}{dt}\,\mathbf{j} \qquad\quad \ddot{\mathbf{r}}(t) = \frac{d^2x}{dt^2}\,\mathbf{i} + \frac{d^2y}{dt^2}\,\mathbf{j}\]

  • The familiar rules survive: \(\dfrac{d}{dt}(\mathbf{r}_1 + \mathbf{r}_2) = \dot{\mathbf{r}}_1 + \dot{\mathbf{r}}_2\), \(\dfrac{d}{dt}(k\mathbf{r}) = k\dot{\mathbf{r}}\), and \(\dfrac{d}{dt}\big(f(t)\mathbf{r}(t)\big) = f'(t)\mathbf{r}(t) + f(t)\dot{\mathbf{r}}(t)\).
  • Gradient of the path at a point: \(\dfrac{dy}{dx} = \dfrac{\dot y}{\dot x}\) (chain rule, \(\dot x \neq 0\)).
Geometry

\(\dot{\mathbf{r}}(t)\) points along the tangent to the path, in the direction of travel. Parallel to \(\mathbf{i}\) ⇔ \(\dot y = 0\); parallel to \(\mathbf{j}\) ⇔ \(\dot x = 0\).

Worked example: tangents from \(\dot{\mathbf{r}}\)

Example 5

\(\mathbf{r}(t) = (t^2 - 1)\,\mathbf{i} + 2t\,\mathbf{j}\), \(t \geq 0\). Find the Cartesian path, \(\dot{\mathbf{r}}(t)\), the gradient of the path at \(t = 1\), and any time the particle moves parallel to \(\mathbf{i} + \mathbf{j}\).

  • Path: \(y = 2t \Rightarrow t = \dfrac{y}{2}\), so \(x = \dfrac{y^2}{4} - 1\) — a sideways parabola with \(y \geq 0\).
  • \(\dot{\mathbf{r}}(t) = 2t\,\mathbf{i} + 2\,\mathbf{j}\); at \(t = 1\), \(\dot{\mathbf{r}} = 2\mathbf{i} + 2\mathbf{j}\).
  • Gradient: \(\dfrac{dy}{dx} = \dfrac{\dot y}{\dot x} = \dfrac{2}{2t} = \dfrac{1}{t}\); at \(t = 1\) the gradient is \(1\) — matching the direction \(2\mathbf{i}+2\mathbf{j}\).
  • Parallel to \(\mathbf{i} + \mathbf{j}\) needs \(\dot y / \dot x = 1\): \(2t = 2 \Rightarrow t = 1\). Only then.

Seeing it: \(\dot{\mathbf{r}}\) rides the curve

At \(t=1\) the particle is at \((0,2)\) and \(\dot{\mathbf{r}} = 2\mathbf{i} + 2\mathbf{j}\) lies along the tangent — gradient \(1\), as computed.

Worked example: rebuilding \(\mathbf{r}\) from \(\ddot{\mathbf{r}}\)

Example 6

A projectile has \(\ddot{\mathbf{r}}(t) = -9.8\,\mathbf{j}\), with \(\dot{\mathbf{r}}(0) = 20\,\mathbf{i} + 15\,\mathbf{j}\) and \(\mathbf{r}(0) = \mathbf{0}\). Find \(\mathbf{r}(t)\).

  • Antidifferentiate once: \(\dot{\mathbf{r}}(t) = -9.8t\,\mathbf{j} + \mathbf{c}_1\). The constant is a vector.
  • \(\dot{\mathbf{r}}(0) = \mathbf{c}_1 = 20\,\mathbf{i} + 15\,\mathbf{j}\), so \(\dot{\mathbf{r}}(t) = 20\,\mathbf{i} + (15 - 9.8t)\,\mathbf{j}\).
  • Antidifferentiate again: \(\mathbf{r}(t) = 20t\,\mathbf{i} + (15t - 4.9t^2)\,\mathbf{j} + \mathbf{c}_2\), and \(\mathbf{r}(0) = \mathbf{0} \Rightarrow \mathbf{c}_2 = \mathbf{0}\).
  • \(\mathbf{r}(t) = 20t\,\mathbf{i} + (15t - 4.9t^2)\,\mathbf{j}\) — Lesson 4 takes this projectile apart.
Skeleton

Two antidifferentiations, two vector initial conditions. Same chain of custody as last week — once per component.

Velocity, acceleration and speed

  • \(\mathbf{v} = \dot{\mathbf{r}}\), \(\mathbf{a} = \ddot{\mathbf{r}}\); speed \(= |\mathbf{v}|\); distance \(= \int |\mathbf{v}|\,dt\).
  • Projectiles: greatest height, range, and what "at the top" really means.
  • Circular motion, and the closest-approach technique.

Velocity, acceleration and speed

  • Velocity \(\mathbf{v}(t) = \dot{\mathbf{r}}(t)\) — a vector, tangent to the path, pointing in the direction of motion.
  • Acceleration \(\mathbf{a}(t) = \ddot{\mathbf{r}}(t)\) — need not point along the path.
  • Speed \(= |\mathbf{v}(t)| = \sqrt{\dot{x}^2 + \dot{y}^2}\) — a scalar, never negative.
Distance travelled

Distance from \(t_1\) to \(t_2\) is the integral of speed:

\[\int_{t_1}^{t_2} \sqrt{\dot{x}^2 + \dot{y}^2}\;dt\]

Usually a CAS integral in practice.

Common trap

"At rest" means both components of \(\mathbf{v}\) are zero. A projectile at its peak is not at rest — only \(\dot y = 0\).

Worked example: projectile

Example 7

\(\mathbf{r}(t) = 20t\,\mathbf{i} + (15t - 4.9t^2)\,\mathbf{j}\) (metres, seconds) — from Example 6. Find the initial speed, the greatest height, and the range.

  • \(\mathbf{v}(t) = 20\,\mathbf{i} + (15 - 9.8t)\,\mathbf{j}\); initial speed \(= \sqrt{20^2 + 15^2} = \sqrt{625} = 25\) m/s.
  • Greatest height when the vertical component of \(\mathbf{v}\) is zero: \(t = \dfrac{15}{9.8} \approx 1.53\) s; height \(y = \dfrac{15^2}{2 \times 9.8} \approx 11.5\) m.
  • Lands when \(15t - 4.9t^2 = 0 \Rightarrow t = \dfrac{15}{4.9} \approx 3.06\) s; range \(x = 20t \approx 61.2\) m.
  • At the top: \(\mathbf{v} = 20\,\mathbf{i}\) — speed \(20\) m/s, not zero. Only \(\dot y\) vanishes.

Circular motion: constant speed ≠ zero acceleration

  • \(\mathbf{r}(t) = a\cos(\omega t)\,\mathbf{i} + a\sin(\omega t)\,\mathbf{j}\): a circle of radius \(a\).
  • \(\mathbf{v} = -a\omega\sin(\omega t)\,\mathbf{i} + a\omega\cos(\omega t)\,\mathbf{j}\); speed \(= a\omega\), constant.
  • \(\mathbf{a} = \ddot{\mathbf{r}} = -\omega^2\,\mathbf{r}\) — nonzero, pointing at the centre.
  • \(\mathbf{r} \cdot \mathbf{v} = 0\): velocity is always perpendicular to position.
Why

Acceleration measures change in the velocity vector. Here the length of \(\mathbf{v}\) is fixed but its direction turns — that turning is the acceleration.

Worked example: closest approach

Example 8

A boat's position (km) is \(\mathbf{r}(t) = (t-3)\,\mathbf{i} + 2t\,\mathbf{j}\), \(t \geq 0\) hours. Find its minimum distance from the origin (a lighthouse).

  • Minimise the square of distance: \(d^2 = (t-3)^2 + (2t)^2 = 5t^2 - 6t + 9\).
  • \(\dfrac{d}{dt}(d^2) = 10t - 6 = 0 \Rightarrow t = 0.6\) h — a minimum (upward parabola).
  • \(d^2 = 5(0.36) - 3.6 + 9 = 7.2\), so \(d = \dfrac{6}{\sqrt{5}} = \dfrac{6\sqrt{5}}{5} \approx 2.68\) km.
Why \(d^2\)?

Same minimiser, no square roots in the derivative. Standard trick — cite it and use it. (Alternative: at closest approach, \(\mathbf{r} \cdot \mathbf{v} = 0\).)

Week 2 summary

  • Paths: eliminate the parameter, respect the domain, note direction and start point.
  • Collisions need equal position at equal time — path intersections don't.
  • Calculus acts componentwise; vector constants of integration need initial conditions; \(\dfrac{dy}{dx} = \dfrac{\dot y}{\dot x}\).
  • \(\mathbf{v} = \dot{\mathbf{r}}\) is tangent; speed is \(|\mathbf{v}|\); distance is \(\int |\mathbf{v}|\,dt\); constant speed can still mean nonzero acceleration.
