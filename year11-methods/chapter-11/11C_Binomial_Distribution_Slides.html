01 · Where the formula comes from

Roll a die 3 times, count the 3s — every path, priced

Two ingredients per value of X

Price of one path: each roll is independent, so multiply: (16)2(56)1 for two 3s and one non-3.
Number of paths: which two rolls are the 3s? Choose 2 of 3 positions: 3C2 = 3 paths — 10C does the counting.
Putting it together
Pr(X = 2) = 3C2 (16)2 (56)1 = 15216

Same-price paths, counted by a selection. That's the whole distribution.

02 · The binomial distribution

Four conditions, one formula

Binomial experiment
  • n identical trials;
  • each trial: two outcomes — success or failure;
  • Pr(success) = p is constant across trials;
  • trials are independent.
Binomial probability

If X = number of successes in the n trials, then

Pr(X = x) = nCx px (1−p)n−x x = 0, 1, …, n

Binomial or not?

Toss a coin 10 times, count heads✓ binomial
Roll a die 60 times, count 6s✓ binomial (success = "a 6")
Draw 5 cards without replacement, count hearts✗ p changes each draw
Ask 15 people which of two cheeses they prefer✓ binomial

"With replacement" is the tell

Replacing (or a huge population, or coin/dice trials) keeps p constant. Sampling without replacement is 11B territory — not binomial.

03 · By hand

I Do Example 1

In Melbourne, the probability of rain on any November day is 0.4, independently of other days. For a given week, find the probability that rain falls:

  1. a on exactly three days
  2. b on the first three days only (and not on the other four).
  • 1Define X. Let X = number of rainy days in the week: binomial, n = 7, p = 0.4.
  • aPr(X = 3) = 7C3 (0.4)3 (0.6)4.
  • 3= 35 × 0.064 × 0.1296 ≈ 0.290.
  • bA specific order — one path only, no 7C3: (0.4)3 (0.6)4 ≈ 0.008. Thirty-five times smaller!

The a-vs-b distinction

"Exactly three days" = any three of the seven → count the ways with 7C3. "The first three days" names one particular path → just multiply the seven probabilities. Read for which one the question wants.

Setting-out habit

Define X in words.
State X ~ binomial, n, p.
Write the formula line before the number.
04 · Using the TI-Nspire CX II CAS

CAS skills Example 2

binomPdf for Pr(X = x), binomCdf for a range Pr(a ≤ X ≤ b). Found under menu ▸ Probability ▸ Distributions.

For the rain model (n = 7, p = 0.4), find the probability that rain falls on:

  1. a no more than three days
  2. b at least three days.
  • a"No more than 3" = Pr(X ≤ 3): binomCdf(7, 0.4, 0, 3) = 0.710.
  • b"At least 3" = Pr(X ≥ 3): binomCdf(7, 0.4, 3, 7) = 0.580.
  • ✓Sense check: the two answers overlap at X = 3, so together they exceed 1: 0.710 + 0.580 = 1.29 = 1 + Pr(X = 3). ✓

Lower and upper bounds

Pr(X ≤ 3)binomCdf(n, p, 0, 3)
Pr(X ≥ 3)binomCdf(n, p, 3, n)
Pr(X < 3)binomCdf(n, p, 0, 2)
Pr(2 ≤ X ≤ 5)binomCdf(n, p, 2, 5)

11A language strikes again

The whole battle is converting "at least / more than / no more than" into the right bounds. Strict inequalities shift the bound by one.

05 · Small n, exact answers

We Do Example 3

A machine has three identical components. Each fails during a year with probability 0.01, independently. Find the probability that, in a year:

  1. a no component fails
  2. b no more than one component fails.
  • 1X = number of failures: binomial, n = 3, p = 0.01.
  • aPr(X = 0) = (0.99)3 = 0.970 (3 d.p.).
  • bPr(X = 1) = 3C1 (0.01)(0.99)2 = 3 × 0.009801 = 0.029403.
  • 4Pr(X ≤ 1) = 0.970299 + 0.029403 = 0.9997 (4 d.p.).

When to stay by hand

For n ≤ 4 or when an exact value is demanded, the formula beats the calculator. Note nC0 = 1 and nC1 = n — the two cases you'll use most.

Don't drop the failure power

Pr(X = 1) ≠ 3 × 0.01. The other two components must not fail: the (0.99)2 is part of the price of the path.

06 · Two exam favourites

We Do Examples 4 and 5

Example 4 The probability of winning a game is 0.25. What is the least number of games that must be played so that Pr(win at least twice) > 0.9?

  • 1Flip it: Pr(X ≥ 2) > 0.9 ⇔ Pr(X < 2) < 0.1.
  • 2Pr(X < 2) = (0.75)n + 0.25n(0.75)n−1 — the x=0 and x=1 terms.
  • 3No algebraic solution — solve numerically on CAS (table or nSolve).
  • 4n = 14 gives 0.101; n = 15 gives 0.080 < 0.1 → least n = 15.

Example 5 An archer scores a maximum with probability 0.4. Out of five shots, find the probability of exactly three maximums, given she scores at least one.

  • 1Pr(X = 3) = 5C3(0.4)3(0.6)2 = 0.2304.
  • 29F: Pr(X = 3 | X > 0) = Pr(X = 3)Pr(X > 0) (X = 3 implies X > 0).
  • 3Pr(X > 0) = 1 − (0.6)5 = 0.92224.
  • 40.23040.92224 ≈ 0.250.

09 · Summary 11C

The binomial distribution — what to remember

The model

  • Binomial ⇔ n identical, independent, success/fail trials with constant p (sampling with replacement).
  • Pr(X = x) = nCx px(1−p)n−x — ways × price of a path.
  • CAS: binomPdf (equals), binomCdf (ranges).

The routine

Define X in words; state n and p.
Translate the inequality (11A skills), then Pdf or Cdf.
"At least once" → 1 − Pr(X = 0) = 1 − (1−p)n. Least n → set up the inequality, solve numerically.

The #1 trap

The nCx coefficient. Leave it off when the order is unspecified and your answer is many times too small; include it when the question fixes a specific order and it's many times too big. One question: does the question name which trials succeed?

Check · Mini-whiteboards

Check Hinge question — everyone shows an answer

A  5C2 (16)2 (56)3
B  (16)2 (56)3
C  5C2 (16)2
D  2 × 16
