01 · The definition

Independence: information that changes nothing

Independent events

Events A and B are independent if knowing one occurred doesn't change the probability of the other:

Pr(A | B) = Pr(A)

Equivalently — and this is the usual test:

Pr(A ∩ B) = Pr(A) × Pr(B)

Physical vs mathematical

Separate coin tosses are physically independent, so they're mathematically independent too. But events can pass the test without any physical separation — always check the numbers.

Warm-up W3, revisited

For two coin tosses, Pr(2nd is H | 1st is H) = 12 = Pr(2nd is H).

The first toss told us nothing about the second. That's independence — and it's exactly why "tails is due" (the gambler's fallacy) is wrong.

Three equivalent checks

Pr(A | B) = Pr(A)
Pr(B | A) = Pr(B)
Pr(A ∩ B) = Pr(A) × Pr(B) — use this one.
02 · Testing from a table

I Do Example 1

500 people were classified by age and social media use. Is regular use of social media independent of age?
  • 1The joint probability: Pr(under 25 ∩ Yes) = 200500 = 0.4.
  • 2The product: Pr(under 25) × Pr(Yes) = 0.48 × 0.6 = 0.288.
  • 30.4 ≠ 0.288, so the events are not independent — age changes the probability of using social media.

Sanity check with conditionals

Pr(Yes | under 25) = 200/240 ≈ 0.83 but Pr(Yes) = 0.6. Knowing the age changed the probability — dependent.

03 · Testing from sets

We Do Example 2

A number is drawn at random from {1, 2, 3, 4, 5, 6, 7, 8}. Let A = {1, 2, 3, 4}, B = {1, 3, 5, 7} and C = {4, 6, 8}.

  1. a Are A and B independent?
  2. b Are A and C independent?
  • 1Pr(A) = 12, Pr(B) = 12, Pr(C) = 38.
  • aA ∩ B = {1, 3}: Pr = 14. Product: 12 × 12 = 14. Equal → independent. ✓
  • bA ∩ C = {4}: Pr = 18. Product: 12 × 38 = 316. Not equal → not independent. ✗

The routine, always

Find Pr(A), Pr(B) and Pr(A ∩ B).
Multiply the first two.
Equal to the joint? Independent. Otherwise not. State the comparison in your answer.

Intuition can't be trusted here

A and B above have no "physical" separation — yet they pass. Independence is a property of the numbers.

04 · Testing and using

We Do Example 3

a Events A and B have Pr(A) = 0.4, Pr(B) = 0.2 and Pr(A ∩ B) = 0.2. Are they independent?

  • 1Product: 0.4 × 0.2 = 0.08.
  • 20.2 ≠ 0.08 → not independent. (In fact Pr(B | A) = 0.5 ≠ 0.2.)

b A family owns a TV with probability 0.75 and a station wagon with probability 0.25, independently. Find the probability that a family owns: i both ii at least one.

  • 1i Independence lets us multiply: Pr(T ∩ S) = 0.75 × 0.25 = 0.1875.
  • 2ii Addition rule: Pr(T ∪ S) = 0.75 + 0.25 − 0.1875 = 0.8125.

Two rules, one question

Independence gives the ∩; the addition rule turns it into the ∪. This pairing is everywhere in Ex 9G.

05 · Don't confuse them

Independent vs mutually exclusive — side by side

Independent

  • Can happen together — one just doesn't change the other's chances.
  • Pr(A ∩ B) = Pr(A) × Pr(B) — usually positive.
  • Example: "first toss H" and "second toss H".

Mutually exclusive

  • Cannot happen together: A ∩ B = ∅.
  • Pr(A ∩ B) = 0, so Pr(A ∪ B) = Pr(A) + Pr(B).
  • Example: "roll a 2" and "roll a 5".
The clincher

If Pr(A) > 0 and Pr(B) > 0 and A, B are mutually exclusive, then Pr(A ∩ B) = 0 but Pr(A) × Pr(B) > 0 — so they fail the independence test. Mutually exclusive events are extreme dependence: if one happens, the other is impossible.

You Do Example 4

a. A and B are independent with Pr(A) = 0.5, Pr(B) = 0.2. Find Pr(A ∪ B).
Pr(A ∩ B) = 0.1; Pr(A ∪ B) = 0.5 + 0.2 − 0.1 = 0.6.
b. A Venn diagram shows counts: A only 9, both 3, B only 1, neither 3. Show that A and B are independent.
n = 16. Pr(A) = 1216 = 34, Pr(B) = 416 = 14; product = 316 = Pr(A ∩ B). ✓
c. Pr(wife watches a show) = 0.5, Pr(husband watches) = 0.4, Pr(wife | husband) = 0.9. Find Pr(both watch) and Pr(husband | wife).
Pr(W ∩ H) = 0.9 × 0.4 = 0.36; Pr(H | W) = 0.36/0.5 = 0.72. (Not independent!)
d. Blood groups are independent between spouses: Pr(O) = 0.5, Pr(B) = 0.1. Find Pr(wife is O and husband is B).
0.5 × 0.1 = 0.05.

CAS skills Independence with an unknown

Harder questions (like Ex 9G Q15) make Pr(A) an unknown and optimise. Let CAS do the algebra.

The setup (Ex 9G Q15)

A and B independent, Pr(A) + Pr(B) = 0.3. Minimise Pr(A ∪ B).

Let Pr(A) = x. Then

Pr(A ∪ B) = x + (0.3 − x) − x(0.3 − x) = 0.3 − 0.3x + x2.

On the calculator

  • Graph f1(x) = 0.3 − 0.3x + x2 for 0 < x < 0.3.
  • menu ▸ Analyze Graph ▸ Minimum: at x = 0.15, value 0.2775.
  • So the minimum is when Pr(A) = Pr(B) = 0.15.
08 · Summary 9G

Independent events — what to remember

Definition & test

  • Independent: Pr(A | B) = Pr(A) — information changes nothing.
  • Test (and use): Pr(A ∩ B) = Pr(A) × Pr(B).
  • Only multiply probabilities when independence is given or shown.

The routine

Compute joint and product separately; compare; conclude with the comparison shown.
Using independence: multiply for ∩, then addition rule for ∪.
From a table: pick one cell, test it against its margins.

The #1 trap

"Mutually exclusive" and "independent" are not synonyms — they're almost opposites. Mutually exclusive means one rules the other out (total dependence). Independent means one tells you nothing about the other.

Check · Mini-whiteboards

Check Hinge question — everyone shows an answer

On your whiteboard write A, B, C or D. On the count of three, hold it up — everyone answers at once.
Events A and B are mutually exclusive, with Pr(A) = 0.3 and Pr(B) = 0.4. Are they independent?
A  No — Pr(A ∩ B) = 0 but Pr(A)×Pr(B) = 0.12
B  Yes — they don't affect each other
C  Yes — because Pr(A ∪ B) = 0.7
D  Can't tell without more information
Answer: A. The test fails: 0 ≠ 0.12. B is the classic confusion — "can't happen together" actually means one completely determines the other's fate; C quotes an irrelevant fact; D — we have all the information we need.
